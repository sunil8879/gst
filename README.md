## ✨ Features
- **Secure Login:** Google Auth integration (No database required).
- **Auto-Generation:** Automatic 8-digit alphanumeric invoice numbers (e.g., INV-8GF4V2PT).
- **Smart Search:** HSN/SAC and GST rates lookup from a built-in database.
- **Invoice History:** Save and manage previous invoices locally in your browser.
- **Multiple Templates:** Switch between Modern, Professional, Retail, and Service styles.
- **Word Export:** Generate professional .doc files directly to your device.
- **Mobile Friendly:** Designed to work perfectly on laptops and smartphones.
